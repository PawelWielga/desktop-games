# Brand icons

Brand icons live here only when the app needs to represent a real external service.

Current assets:

- `youtube.svg` - YouTube brand icon, based on the Simple Icons YouTube SVG path and adjusted to the official red/white mark style.

## Rules

- Keep brand icons as local SVG files instead of hotlinking remote images.
- Do not redraw brand icons in a way that changes their identity.
- Check the brand owner's usage guidelines before adding a new brand icon.
- Keep `icon` in `AppRegistry` as a text fallback and use `iconAsset` for the real SVG.

Brand icons can be distributed under open icon-set licenses, but the brand name and logo may still be protected as trademarks.
